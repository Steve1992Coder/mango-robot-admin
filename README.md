# robot-admin

Vue3+Vite+Typescript+Element-plus+Tailwindcss+Axios+mock.js

## 命令

### 安装依赖

```sh
npm install
```

### 开发启动

```sh
npm run dev
```

### 生产构建

```sh
npm run build
```

### 单元测试 [Vitest](https://vitest.dev/)

```sh
npm run test:unit
```

### 代码规范 [ESLint](https://eslint.org/)

```sh
npm run lint
```
